01 Abstract

This system implements a novel hybrid quantum-classical machine learning pipeline for medical image classification. We combine ResNet-50 deep learning feature extraction with a Quantum Support Vector Classifier (QSVC) to achieve enhanced binary classification of endoscopy images for Ulcerative Colitis diagnosis. The quantum kernel leverages entanglement and superposition to map features into high-dimensional Hilbert spaces, potentially capturing complex patterns missed by classical methods.

03 Feature Extraction: ResNet-50

We use ResNet-50, a 50-layer deep residual network pre-trained on ImageNet, as our feature extractor. The network's ability to learn hierarchical features makes it ideal for medical image analysis.

Why ResNet-50?

  • ‚Ä¢ Skip connections prevent vanishing gradients
  • ‚Ä¢ Pre-trained on 1M+ images (transfer learning)
  • ‚Ä¢ Captures low to high-level features
  • ‚Ä¢ 2048-dimensional rich feature vectors

Processing Steps

  • 1. Resize image to 224√ó224
  • 2. Normalize with ImageNet mean/std
  • 3. Forward pass through ResNet
  • 4. Extract avgpool layer (2048-D)
