# UC Prediction QML

Hybrid Quantum-Classical machine learning system for Ulcerative Colitis prediction.

## Standard Features
- Single Image Diagnostic
- Batch Processing
- CSV Clinical Data Analysis
- Quantum Circuit Visualization

## Installation
1. Install requirements: `pip install -r backend/requirements.txt`
2. Start backend: `uvicorn backend.main:app --reload`
3. Start frontend: `python -m http.server 3000`
